This project is part of the Analysis and Design of Algorithms (ADA) Lab Assignment – 1. It focuses on understanding and analyzing different algorithm complexities through practical implementation in Python.

The assignment includes:

Task 1: Implementation and comparison of time complexities (O(1), O(n), O(n²), O(log n)) using execution time and memory usage.
Task 2: Comparison between Linear Search and Binary Search based on performance.
Task 3: Analysis of recursion using Factorial and Fibonacci (both recursive and dynamic programming approaches).
Task 4: Solving recurrence relations through recursive functions and analyzing their behavior.

Graphs are plotted to visualize time and space complexity, helping in better understanding of algorithm efficiency.
